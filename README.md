# 🏠 Muebles BR - Sitio Web Profesional

Sitio web profesional para **Home BR Muebles**, empresa especializada en fabricación de **muebles a medida, remodelación y diseño de interiores** en Santiago, Chile.

## 📋 Descripción del Proyecto

Muebles BR es una empresa dedicada a la creación de muebles personalizados de alta calidad, remodelación integral de espacios y diseño de interiores. Este sitio web actúa como vitrina digital y generador de leads para los servicios de la empresa.

## 🎯 Características Principales

✅ **Diseño Responsivo** - Funciona perfectamente en escritorio, tablet y móvil  
✅ **Galería de Proyectos** - Muestra los trabajos realizados  
✅ **Sección de Servicios** - Describe los servicios ofrecidos  
✅ **Formulario de Contacto** - Facilita la comunicación con clientes  
✅ **Botón WhatsApp Flotante** - Acceso rápido a WhatsApp Business  
✅ **Optimizado para SEO** - Meta tags y estructura semántica  
✅ **Animaciones Suaves** - Experiencia de usuario mejorada

## 🚀 Cómo Usar

1. Clonar el repositorio:
```bash
git clone https://github.com/abogadarayenaraya-dotcom/muebles-br.git
```

2. Abrir `index.html` en el navegador

3. (Opcional) Usar un servidor local:
```bash
python -m http.server 8000
# o con Node.js
npx http-server
```
